# Mauro Sebastião — Portfolio Site

## 🚀 Publicar no Netlify (Grátis)

### Passo 1 — Criar conta
Aceda a https://www.netlify.com e crie uma conta gratuita (pode usar a conta GitHub/Google).

### Passo 2 — Fazer upload
1. No painel Netlify, clique em **"Add new site"** → **"Deploy manually"**
2. Arraste a pasta `portfolio` (que contém o `index.html`) para a área de upload
3. O site fica online em segundos com um URL como: `https://mauro-sebastiao.netlify.app`

### Passo 3 — Domínio personalizado (opcional)
- Em Site Settings → Domain Management, pode adicionar um domínio personalizado
- Ou usar o subdomínio gratuito que o Netlify atribui

## ✅ Funcionalidades

- **Portfólio completo** com todas as informações do CV
- **Painel Admin** para editar qualquer conteúdo
- **Upload de foto** de perfil
- **Galeria** de fotos e vídeos dos projetos
- **Seletor de idioma** PT / EN
- **Seletor de cores** com 7 temas + cor personalizada
- **Gerador de CV** em formato PDF (modelo europeu)
- **Gerador de Carta de Apresentação** PT / EN
- **Formulário de contacto** via email
- **Dados guardados** no localStorage do browser
